# php-utils

# 项目说明
此项目提供PHP 常用功能集合，主要功能包括：

| 功能            | 说明           | 版本  |
| ------------- |:-------------:| -----:|
| 数组操作         | 无          | \>=0.1 |
| 字符串操作      | 无            |   \>=0.1 |


# 版本说明
当前请使用 dev-master 版本

```composer require "woodw/php-utils:dev-master"```

# 测试

* 运行方式1：(命令行方式)
```bash
./phpunit func/ArrayTest.php
```
或 
```bash
./phpunit --bootstrap BaseTestCase.php func/ArrayTest.php 
```

* 运行方式2：（通过 phpunit.xml 运行）  
 1. 配置 phpunit.xml
 2. 通过 phpunit.xml 运行
 ```
./phpunit
```
