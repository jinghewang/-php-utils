# php-utils

# 项目说明
此项目提供PHP 常用功能集合，主要功能包括：

| 功能            | 说明           | 版本  |
| ------------- |:-------------:| -----:|
| 数组操作         | 无          | \>=0.1 |
| 字符串操作      | 无            |   \>=0.1 |
| 日志操作          | 无         |    \>=0.1 |


# 版本说明
当前请使用 dev-master 版本

```composer require "woodw/php-utils:dev-master"```
